In my arrival as an intern, the foundry sector was passing to a great change on his Rebar process, needing a new system for generate and print the rebar rolls tags and with an imense queue on I.T. department they ask me for help. During some weeks i developed a VBA application that consumes a database and from that generate and print rebar rolls tags besides save an activity registry. It is used until nowdays.

QA Cetification System

With a need very similar to the one above. PPC team ask me if i could do an system thats validate and print QA certificates. It was a much more complex and problem, since the quality validation depends on numerous factors and i had to bring that into the system. The systems test each register on a database that corresponds to a rebar roll and sees if it have any problem in its chemical composition and mechanical proprieties according to a specification table (wich varites into 14 rebar types and their own specifications). If succeed the system generate and print a QA certificate for the register, if not it will save the register on a "not ok" sheet and send it to the responsable process technician by email. Like the Tag Generator, it is used until nowdays.
